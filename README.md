
# Ramu Kaka AI 🤖

Dilli ka original chatbot — witty, funny, and full of jugaad.

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Create a `.env` file and add your Groq API key (get one free at https://console.groq.com):
   `GROQ_API_KEY=your_key_here`
3. Run the app:
   `npm run dev`
